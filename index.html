AZ Odometer Statement Generator














Printing Instructions

1. Enter Required Information:

  • Customer Name
  • VIN
  • Employee Name (leave blank if printing a future ODO)

2. Date Settings:

  • The date is automatically set to the current date.
  • If printing a future ODO, clear the date field.

3. Generating the Print Preview:

  • Click the Generate button.
  • A new page will open with the print preview.

4. Printer Selection & Quality Settings:

  • Select EPSON LX 350+ as the printer.
  • Ensure the print quality is set to Max DPI: (240x144 or 120x77)

5. Paper Guide & Adjustments:

  • For Chromebook Users:
    • Set the blue guide on the left to the "10" marker.
    • Apply slight pressure when inserting the ODO sheet.
  • For ThinkCentre (Windows) Users:
    • Set the blue guide on the left to the "9.4" marker.
    • (You should see six small tally marks on the right side of the 10.)
    • Apply slight pressure when inserting the ODO sheet.
    • Press the Micro-Adjust DOWN arrow once.
